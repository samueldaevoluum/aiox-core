## 🎯 PRESCRIÇÃO AUTOMÁTICA (Dia 0)

Sistema gera **tarefas prescritas** baseado no nível:

```yaml
PRESCRIÇÃO — João (Startup ABC)

Foco 1: FINANCEIRO (N0.8 → N2)
  Tarefa 1: "Criar planilha de fluxo de caixa mensal" (1 dia)
            └─ Template fornecido: cash-flow-template.xlsx
  
  Tarefa 2: "Definir meta de faturamento mínimo mensal" (1 dia)
            └─ Como: "Qual é seu custo fixo? Aí tem meta."
  
  Tarefa 3: "Fazer reunião de revisão financeira (semanal)" (1 dia)
            └─ Template: 15 min, 3 pessoas

  PRAZO: 3 semanas
  PRÓXIMO NÍVEL: N2 (Organização Inicial)

---

Foco 2: PESSOAS (N0.8 → N2)
  Tarefa 1: "Documentar papel de cada pessoa" (1 dia)
            └─ Template: role-description-tmpl.md
  
  Tarefa 2: "Fazer reunião 1:1 com cada um" (3 dias)
            └─ Script: O que tá funcionando? Bloqueios?
  
  Tarefa 3: "Comunicar expectativas claramente" (1 dia)
            └─ Reunião: "Eis o que espero de cada um"

  PRAZO: 3 semanas
  PRÓXIMO NÍVEL: N2

---

Foco 3: VENDAS (N0.8 → N2)
  Tarefa 1: "Definir pipeline de vendas (3 estágios)" (1 dia)
            └─ Estágios: Prospect → Proposta → Fechado
  
  Tarefa 2: "Criar meta mensal mínima de vendas" (1 dia)
            └─ Meta: 5 clientes/mês ou R$10k MRR?
  
  Tarefa 3: "Reunião semanal de vendas (30 min)" (1 dia)
            └─ Agenda: Quantos na pipeline? Taxa conversão?

  PRAZO: 4 semanas
  PRÓXIMO NÍVEL: N2
```

## 💬 INÍCIO DO COACHING (Dia 1)

### Chat 1: Financeiro

```
João: "Tô muito preocupado com fluxo de caixa. 
       Nunca sei se vou ter dinheiro no fim do mês."

IA: "Entendo essa ansiedade. Vamos começar por aqui.
     Primeiro: você sabe quanto precisa de receita mínima 
     cada mês pra cobrir os custos fixos?"

João: "Mais ou menos... uns 30 mil? Mas não tenho certeza."

IA: [COLETA EVIDÊNCIA]
    └─ Percepção vaga de custo fixo = N0/N1
    └─ Refina F-01 (Controle Financeiro) de 1 → 0.5
    
IA: "Vamos estruturar isso. Você tem onde anotar 
     todos seus custos mensais fixos? Aluguel, folha,
     fornecedores, etc."

João: "Não, tá espalhado em vários lugares."

IA: [COLETA EVIDÊNCIA ADICIONAL]
    └─ Sem visibilidade de custos = confirmação N1
    └─ Prescreve: criar planilha de custos fixos

IA: "Ok, primeira tarefa é lista todos os custos fixos.
     Leva 30 min hoje. Template aqui: [link]
     Amanhã converso com você sobre."
```

**Refinamento Automático após Chat 1:**
```
F-01 (Controle Financeiro):
├─ Resposta Inicial: 1 (Reativo)
├─ Evidência Chat 1: "Não sabe quanto precisa de receita"
├─ Novo Score: 1.0 (confirmado)
└─ Confiança: 80% (antes era 76%)
```

### Chat 5: Operação

```
João: "A galera tá reclamando que não sabe direito 
       qual é o processo de desenvolvimento. Tá tudo 
       na minha cabeça."

IA: [COLETA EVIDÊNCIA]
    └─ "Tudo na minha cabeça" = N0/N1
    └─ Refina O-01 (Documentação) de 1 → 0.8
    
IA: "Entendo. Como vocês sabem que tá pronto pra 
     entregar um projeto?"

João: "Ah, quando eu reviso e acho que tá bom."

IA: [COLETA EVIDÊNCIA]
    └─ Revisão manual = N1
    └─ Refina O-03 (Qualidade) de 2 → 1.5
    
IA: "Vamos documentar o processo. Não precisa ser 
     perfeito. Tipo: qual é a ordem de passos?"

João: "Design → Dev → Testing → Deploy. Eu reviso cada um."

IA: "Ótimo, esse é seu pipeline. Vamos transformar 
     isso em checklist pra equipe usar."
```

**Refinamento após Chat 5:**
```
Operação refinada:
├─ O-01: 1 → 0.8 (confirmação com evidência)
├─ O-03: 2 → 1.5 (refinamento)
├─ NOVO SCORE OPERAÇÃO: (0.8+2+1.5+1+1)/5 = 1.26
└─ Confiança: 85% (subiu de 82%)
```

### Chat 10: Pessoas

```
João: "Meu problema maior é que pessoal tá desmotivado.
       Um saiu mês passado, outro tá com um pé atrás."

IA: [COLETA EVIDÊNCIA]
    └─ Alto turnover = N0/N1 em retenção
    └─ Refina PE-03 (Retenção) de 1 → 0
    
IA: "Quando foi a última vez que você sentou 
     com alguém e perguntou o que tá insatisfeito?"

João: "Não faço isso..."

IA: [COLETA EVIDÊNCIA]
    └─ Sem 1:1s = N0
    └─ Refina PE-02 (Feedback) de 0 → 0 (confirmado)
    
IA: "Ok, vamos começar com 4 reuniões 1:1 
     (uma com cada pessoa). 30 min cada.
     
     Roteiro:
     1. 'Como vai no trabalho?'
     2. 'O que tá funcionando bem pra você?'
     3. 'O que tá frustrante?'
     4. 'O que você quer aprender/crescer?'
     
     Isso é feedback real, não review."

João: "Faz sentido. Vou agendar essa semana."
```

**Refinamento após Chat 10:**
```
Pessoas refinada:
├─ PE-02: 0 → 0 (confirmado)
├─ PE-03: 1 → 0 (evidência forte)
├─ NOVO SCORE PESSOAS: (1+0+0+1+1)/5 = 0.6
└─ Confiança: 80% (subiu de 70%)
```
