# Task: sales-strategy

**Command:** `*sales-strategy {empresa} [--stage {discovery|early|scaling}]`

**Agent:** Alfredo Soares Clone

**Input:** Empresa/startup + contexto mercado

**Output:** Ecosystem Canvas estruturado + roadmap 90 dias

---

## Ecosystem Canvas — 7 Elementos

Quando executar esta task, use o template abaixo e preencha cada elemento:

| Elemento | O que é | Exemplo |
|----------|---------|---------|
| **Canal** | Como você chega no cliente | LinkedIn, SDR, partnership, content |
| **Produto** | O que você vende | SaaS, consultoria, solution |
| **Persona** | Quem compra | CTO, CFO, diretor área |
| **Conteúdo** | O que você ensina antes de vender | 3-5 temas relevantes |
| **Experiência** | Como o cliente se sente | onboarding, suporte, comunidade |
| **Conversão** | Como você converte | proposta, trial, partnership |
| **Retroalimentação** | Como você aprende | NPS, churn, feedback loops |

---

## Execução

1. **Diagnóstico** (20 min)
   - Qual é o stage atual? (discovery/early/scaling)
   - Qual elemento está mais fraco?
   - Qual gera mais impacto agora?

2. **Canvas Preenchido** (30 min)
   - Preencher cada elemento com realidade atual
   - Identificar 3 gaps críticos

3. **Roadmap 90 dias** (30 min)
   - Priorizar elemento por ROI
   - 3 sprints de 30 dias
   - Ações concretas por sprint

4. **Entrega** (10 min)
   - Documento estruturado
   - Owner de cada elemento definido
   - Metrics para medir sucesso

**Tempo total:** ~1.5h

---

## Elicitação (usuario precisa responder)

Faz essas perguntas antes de preencher o canvas:

1. "Qual é o seu maior gargalo agora?" (canal? conversão? produto?)
2. "Qual elemento você controla 100% e qual depende de terceiros?"
3. "Se tivesse que escolher 1 elemento para melhorar nos próximos 90 dias, qual seria?"
