# Preview Proposal Before Sending

## Purpose

Mostrar preview da proposta gerada para validacao da Pati antes do envio ao candidato.

## Process

1. Carregar proposta gerada para o candidato
2. Exibir todos os campos preenchidos
3. Destacar: logos GPTW, valor, cargo, data inicio
4. Solicitar aprovacao: "Proposta aprovada? (SIM/NAO)"
5. Se SIM: marcar como aprovada
6. Se NAO: solicitar correcoes especificas
