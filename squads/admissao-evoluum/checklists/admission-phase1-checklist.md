# Checklist: Admission Phase 1 - Recepcao

## Pre-conditions

- [ ] Dados do candidato recebidos via Asana (nome, cargo, salario, time, email, telefone, gestao direta)
- [ ] Tarefa criada pela Manu no projeto Admissao

## E01 - Proposta

- [ ] Proposta gerada com dados corretos (nome, cargo, valor, data)
- [ ] Lista de beneficios completa (11 itens)
- [ ] Logos GPTW posicionadas conforme regras
- [ ] Valor por extenso confere com valor numerico
- [ ] Pati validou e aprovou a proposta
- [ ] Proposta enviada ao candidato via Gmail

## E02 - Aceite

- [ ] Candidato respondeu aceitando a proposta

## E03 - Boas-Vindas

- [ ] E-mail de boas-vindas enviado com template correto
- [ ] Link do formulario Asana incluido e funcional
- [ ] Data limite informada (24h)

## E04 - Formulario

- [ ] Cooperado preencheu o formulario Asana
- [ ] Dados extraidos e estruturados em 3 blocos
- [ ] Campos obrigatorios validados
- [ ] Certificado escolar recebido (upload)
- [ ] Consentimento LGPD confirmado
- [ ] Sobrenome preferido para e-mail registrado
- [ ] Estado atualizado para PHASE_2_READY
